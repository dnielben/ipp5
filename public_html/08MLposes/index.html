Machine Learning

¿Pueden pensar las máquinas?

En 1945 Alan Turing conjeturó que se podría construir un computador que pensara y dio inicio al campo de desarrollo que hoy llamamos inteligencia artificial

Aunque aún estamos lejos de lograr crear una máquina que razoné y aprenda como los humanos, la humanidad ha avanzado muchos en construir algoritmos que imitan actividades que son atribuidas a la inteligencia humana, como por ejemplo la detección de patrones. El campo que estudia los algoritmos automáticos que aprenden y mejoran cuando se les alimentan datos se denomina Machine Learning.

Esta es un sesión experimental que depende de otros servicios en internet que pueden cambiar con el tiempo. Por lo tanto si usa este sitio sin supervisión tenga en cuenta que los códigos pueden dejar de funcionar con el tiempo.

Siga las instrucciones del profesor para entrenar el modelo.

Para este taller debe construir un modelo usando la herramienta Teachable Machine. Luego debe crear un programa que use el modelo para mover un cursor por un canvas de P5.

Recuerda: siempre puedes restaurar el programa original oprimiendo el botón Restaurar.
