Recursion

La recursión es una forma de definir funciones en la que dicha función es utilizada repetitivamente en su propia definicion para resolver un problema en varios problemas mas pequeños y del mismo tipo. Las definiciones matemáticas normalmente están definidas de forma recursiva. Por ejemplo, la serie de Fibonacci se define recursivamente. La recursión en Haskell es importante porque se realizan cálculos declarando la definicion de algo, en lugar de definir como obtener algo. Por este motivo no hay bucles while o for en Haskell y en su remplazo tenemos que usar la recursión.

Vamos a ver un ejemplo sencillo de recursion en Haskell que recibe un parametro numerico y luego llame en forma recursiva con el valor del parámetro menos 1, para esto debemos tener ecuenta el caso base cuando el parametro sea igual a 0 y el caso inductivo cuando el parametro sea mayor a 0
Nombre_funcion x = if x==0 then x else Nombre_funcion(x-1)
