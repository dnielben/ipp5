Haskell

Declarando e inicializando variables.

Para almacenar valores en memoria los lenguajes de programación usan variables que pueden ser de tipo numérico, booleano, cadena o estructuras de datos.
Una variable se define en Haskell de la siguiente manera:
Nombre_de_la_variable = Valor inicial de la variable
Para poder visualizar en cosola el contenido de la variable se utiliza la funcion show(Nombre_de_la_variable)

Operaciones entre variables

Haskell permite hacer operaciones aritméticas, booleanas y algunas otras funciones predefinidas por Haskell

Operaciones Aritméticas

En Haskell se pueden realizar todas las operaciones aritméticas básicas (+, -, * , /, ^):
Suma: x+y
Resta: x-y
Multiplicación: x*y
División(cociente x entre y): x/y
Potencia(x elevado a y): x^y

Operaciones Booleanas

Estas operaciones pueden devolver dos constantes booleanas que pueden ser True o False

Los siguientes operadores se utilizan entre dos variables booleanas (&&, ||, /= , ==):
AND(conjunción de x e y): x&&y
OR(disyunción de x e y): x||y
NOT(negación logica de x): not x

Los siguientes operadores se utilizan entre dos variables numericas (<, >, <=, >=, ==, /= ):
Menor que(x menor que y): x<y
Mayor que(x mayor que y): x>y
Menor o igual que(x menor o igual que y): x<=y
Mayor o igual que(x mayor o igual que y): x>=y

Los operadores == y /= se utilizan tanto para variables booleanas como variables numéricas y cadenas:
Diferencia de variables(verifica si x es diferente de y): x/=y
Igualdad de variables(verifica si x es igual a y): x==y

Operaciones Predefinidas

El redondeo de x al entero más cercano: round x
El resto de x entre y: mod x y
Máximo entre dos números: max x y
Mínimo entre dos números: min x y
División entera de x entre y: div x y
Absoluto de un número: abs x
Verifica si el número es par: even x
Verifica si el número es impar: odd x
El máximo común divisor de dos números: gcd x y
